# hr-attrition-dashboard-powerbi
| File Name             | Description                                                                                                                                        |
| --------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------- |
| Employee.csv          | Core employee dataset containing demographics, job role, department, overtime status, job satisfaction, performance rating, and attrition details. |
| EducationLevel.csv    | Lookup table defining employee education levels used to categorize education-related attrition trends.                                             |
| PerformanceRating.csv | Contains performance rating data linked to employees for analyzing performance-based attrition patterns.                                           |
| RatingLevel.csv       | Maps performance rating codes to readable rating levels for clearer analysis and reporting.                                                        |
| SatisfiedLevel.csv    | Defines job satisfaction levels to support satisfaction-based attrition analysis.                                                                  |
