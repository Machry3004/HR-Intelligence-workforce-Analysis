# Project Workflow

## Data Ingestion
- Workforce HR data sourced in Excel format
- Loaded into Power BI for analysis
- Initial validation and profiling performed

## Data Cleaning
- Duplicate employee records removed
- Standardized categorical values
- Missing values reviewed and handled
- Data types corrected for analysis

## Data Transformation
- Workforce metrics calculated
- Experience and tenure buckets created
- Attrition indicators derived

## Visualization
- Interactive workforce attrition dashboard
- KPI cards for workforce stability
- Department and role-based analysis
- Slicers for demographic filtering

## Insights
- Workforce attrition concentrated in specific segments
- Early tenure employees show higher exit rates
- Workforce experience correlates with retention
